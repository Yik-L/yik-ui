# 快速开始

## 安装

```sh
npm i @yik_l/ui
```

## 用法

### 完整引入

不用在乎依赖包的大小，没有多少 KB。 大胆引入 😂

```javascript
import { createApp } from "vue";
import App from "./App.vue";
import YikUi from "@yik_l/ui";
const app = createApp(App);
app.use(YikUi);
app.mount("#app");
```

## 开始使用

现在你可以启动项目了。 具体每个组件的使用方法, 请查阅 [每个组件的独立文档](./max-view.md).
