# 大屏适配组件

大屏适配解决方案.

```vue
<template>

</template>
<script setup></script>
```

## 属性

| **属性名** | **说明**   | **类型** | **默认值** |
| ---------- | ---------- | -------- | ---------- |
| width      | 宽度       | string   | 1920px     |
| height     | 高度       | string   | 1080px     |
| isCover    | 是否自适应 | boolean  | false      |
