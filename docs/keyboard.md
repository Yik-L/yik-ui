# 监听键盘

监听移动端键盘是否弹起,无法做组件演示。

## 示例代码

```vue
<template>
  <YikIsKeyboard>
    <p>组件</p>
  </YikIsKeyboard>
</template>
<script setup></script>
```

## 属性

| **属性名** | **说明**                       | **类型** | **默认值** |
| ---------- | ------------------------------ | -------- | ---------- |
| isHideSlot | 键盘弹起时，会隐藏插槽中的内容 | Boolean  | true       |
