# yik-cli

脚手架模板

```sh
npm i yik-cli -g
```

## 用法

安装好以后,直接打开管理员终端,初始化工程

yik init 模板的名称 文件夹名称

```sh
C:\Users\admin> yik init vue3-vite-pc-h5 wjm
 模板下载成功

```

yik 查看帮助

```sh
C:\Users\admin> yik
Usage: main [options] [command]
Options:
  -h, --help      display help for command
Commands:
  init <模板> <名称>
  list            查看所有可用模板
  help [command]  display help for command
```

yik list 查看所有模板

```sh
C:\Users\admin> yik list
vue2-h5
vue2-pc
vue2-uniapp
vue2-electron
vue3-h5
vue3-pc
vue3-vite-pc-h5
vue3-vite-pc-h5_ts

```
